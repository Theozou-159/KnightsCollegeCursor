# Knights College Cursor v1.1.1

This update splits the Diderich cursor into clearly named size-specific Windows
downloads.

## Diderich Downloads

- `DiderichCursor-v1.0.0-Windows-1080p.zip` - smallest Diderich build.
- `DiderichCursor-v1.0.0-Windows-2K.zip` - medium build for 1440p / 2K screens.
- `DiderichCursor-v1.0.0-Windows-4K.zip` - original full-size Diderich build for 4K / high-DPI screens.
- `DiderichCursor-v1.0.0-Windows-Large105.zip` - slightly larger than the original.
- `DiderichCursor-v1.0.0-Windows-XL110.zip` - largest Diderich test build.

The 1080p, 2K, Large105, and XL110 packages install as separate cursor schemes,
so you can test them side by side:

- `Diderich Cursor 1080p`
- `Diderich Cursor 2K`
- `Diderich Cursor Large`
- `Diderich Cursor XL`

The 4K package is the original Diderich build and installs as `Diderich Cursor`.

## Theo

Theo is unchanged in this release:

- `TheoCursor-v1.0.0-Windows.zip`

## Install

1. Download the ZIP that matches the size you want.
2. Extract it.
3. Run the install `.bat` file inside the extracted folder.
4. Choose the scheme manually in Mouse Properties, or run the optional apply
   `.bat` to switch right away.

The installers register the scheme without changing your current cursor by
default. They also save a cursor scheme-list backup to:

```text
%LOCALAPPDATA%\KnightsCollegeCursor\CursorSchemeBackups
```

## Checks

- All release ZIPs were tested with `unzip -t`.
- The Diderich size variants were checked on dark backgrounds.
- The enlarged Diderich variants had visible green edge fringe cleaned before
