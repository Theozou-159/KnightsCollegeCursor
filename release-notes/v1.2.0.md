# Knights College Cursor v1.2.0

Paul and Celio are here.

This update adds two new fan-made Windows animated cursor packs to the Knights
College Cursor collection. The project now includes Diderich, Theo, Paul, and
Celio.

## New Downloads

- `PaulCursor-v1.0.0-Windows-All-Sizes-by-Neilzou1.zip`
- `CelioCursor-v0.1.0-Windows-All-Sizes-by-Neilzou1.zip`

Each ZIP contains these ready-to-install folders:

- `Default`
- `1080p`
- `2K`
- `4K`
- `Large105`
- `XL110`

Open the size you want, then run the install `.bat` inside that folder.

## What Is New

- Paul gets a warm, soft animated cursor pack built around his red-scarf look.
- Celio gets a calm green animated cursor pack with glasses, books, little
  plants, and professor-like gestures.
- The project README now shows all four Pointer previews at the top, so the
  cursor-pack idea is visible immediately.
- Paul and Celio size folders use separate Windows cursor scheme names and
  install folders, so different sizes can be tested side by side.

## Existing Downloads

Diderich and Theo are still available from earlier releases:

- Diderich size-specific ZIPs from v1.1.1
- `TheoCursor-v1.0.0-Windows-by-Neilzou1.zip`

## Install

1. Download the character ZIP.
2. Extract it.
3. For Paul or Celio, open the size folder you want.
4. Run `Install-*-Cursors.bat`, or run `Apply-*-Cursors.bat` if you want to
   install and switch immediately.

Installers register the scheme without changing your current cursor by default.
They also save a cursor scheme-list backup to:

```text
%LOCALAPPDATA%\KnightsCollegeCursor\CursorSchemeBackups
```

## Checks

- The two new release ZIPs were tested with `unzip -t`.
- Paul and Celio all-size packages contain expanded size folders, not ZIP files
  nested inside ZIP files.
- Pointer preview GIFs were added to the project page for Paul and Celio.
