## Install

1. Download the character ZIP.
2. Extract it.
3. If the ZIP has size folders, open the size you want.
4. Run `Install-*-Cursors.bat`, or run `Apply-*-Cursors.bat` if you want to
   install and switch immediately.

Installers register the scheme without changing your current cursor by default.
They also save a cursor scheme-list backup to:

```text
%LOCALAPPDATA%\KnightsCollegeCursor\CursorSchemeBackups
```
